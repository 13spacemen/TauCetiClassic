Портированный с tg аирлок паинтер, который у нас уже не только аирлок паинтер.
"airlock_painter" в пути осталось по наследству. И в общем-то этот момент не
так важен.
При портировании и модификации появился модификатор стоимости покраски cost,
дефолтное значение, выбранное для аирлоков - 10, для окон и стеклянных 
дверей при вызове функции используется параметр 1.

Части кода взаимодействия с окружением лежат в:
ode\game\machinery\doors\airlock.dm - о да, аирлок паинтер с tg умеет красить 
двери!
code\game\machinery\doors\windowdoor.dm - и еще перекрашивать стеклянные
(особенность местной модификации).
code\game\objects\structures\window.dm - еще одна особенность, умеет 
перекрашивать окна.
